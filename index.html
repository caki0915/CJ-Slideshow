Example on how you can use the slideshow

Getting started

Include cj_slideshow.less, cj_slideshow.js, LESS and JQuery in header. You can either download them and include them in your website or link them through a CDN as in the example below.

<!-- JQuery on Google CDN -->
<script src="http://ajax.googleapis.com/ajax/libs/jquery/1.10.2/jquery.min.js"></script>
<!-- CJ Slider plugin -->
<script src="js/cj_slideshow.js"></script>
<!-- CJ Slider less stylesheet -->
<link href="less/cj_slideshow.less" type="text/css" rel="stylesheet/less"> 
<!-- LESS on CloudFlare CDN -->
<script src="http://cdnjs.cloudflare.com/ajax/libs/less.js/1.4.1/less.min.js"></script>

Javascript:

$(document).ready(function(){
    'use strict'
    $('#slideshow_1').slideshow();
});

Styling

The element can be styled in different ways. In the top of the LESS stylesheet you have a couple of variables you can tweak. I you want to style it even further you can scroll down the LESS file customize. 

@slideshow_width:400px; //Width of the slideshow
@slideshow_height:300px; //Height of the slideshow
@slideshow_caption_height: 120px; //Height of the black caption-plane
@slideshow_caption_bottom_space: 30px; //How far the caption-plane is from the bottom
@slideshow_caption_opacity: 0.7; //Opacity of the caption-plane
@slideshow_margin: 10px; //The margins of the slideshow

Optional parameters

You can pass in an object as a parameter in the plugin with different settings that affect the behaviour of the slideshow. Every setting has a default value and is optional.

NameDefault valueDescription
width400The height for the slideshow in pixels
height300The width for the slideshow in pixels
interval3000Interval between transtitions in milliseconds
speed1000Speed of the fade-out effect in milliseconds
looptrueTrue or false, If true, the slideshow will loop after last slide.
startIndex00-based index of the first frame in the slideshow
elementsfigureThe element to slide
